# Grypto
Terminal Application to check price of Cryptocurrencies

## Installation

```sh
$ go get github.com/gostudent/Grypto
$ go install github.com/gostudent/Grypto
```

## Usage

Make sure you have set $GOPATH and $GOPATH/bin is in $PATH

`$ Grypto`
